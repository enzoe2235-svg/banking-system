# Banking System (Python CLI Project)
📌 Overview

This is a Command-Line Interface (CLI) based Banking System developed using Python.
The application allows users to create accounts, securely log in, and perform basic banking operations such as deposit, withdrawal, and balance checking.

This project demonstrates the use of Object-Oriented Programming (OOP) concepts and basic input handling in Python.



# Features
*✅ Account creation using email
*✅ Automatic username generation
*✅ Secure login system (username & password)
*✅ Deposit money
*✅ Withdraw money with balance validation
*✅ Check account balance
*✅ Input validation and error handling


# Concepts Used
*Object-Oriented Programming (Classes & Objects)
*Encapsulation
*Dictionary for data storage
*Conditional statements
*Exception handling (try-except)
*User input handling


# How It Works
1.) User creates an account using email and password
2.) Username is automatically generated
3.) User logs in using username and password
4.) After login, user can:
 *Deposit money
 *Withdraw money
 *Check balance
